# 0501_HW_LinuxLab_UtilityScript

Simple script designed to copy files, store backups and create the necessary directories to complete the process.
Afterwards the script will give a report with the date time and, other basic info.
Script Actions

Creates directories
mkdir -p
Creates sample files
touch
Writes text into files
echo >
Lists directory contents
ls -al
Copies files to Backup
cp -v
Moves and renames file
mv -v
Generates report
cat
Displays report


Make the lab executable: chmod +x 0501_HW_LinuxLab_UtilityScript.sh
Luanch the script : bash 0501_HW_LinuxLab_UtilityScript.sh
